# NTFS
Nintendo Tile Format Screen (NTFS)

| Offset | Length | Name        | Description                                                                                                          |
|--------|--------|-------------|----------------------------------------------------------------------------------------------------------------------|
| DATA   | 0x2    | Screen Data | Indexes to a tile. Format is (YYYYXXNNNNNNNNNN). Y4 Palette Number. X2 Transformation (YFlip/XFlip). N10 Tile Number |
